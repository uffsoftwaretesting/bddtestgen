User Story 2: Forgot Password
Description:
As a registered user who has forgotten their password, I want to be able to reset it so that I can regain access to my account.
Screen Title: Forgot Password
The form must contain the following fields:
Email (inputText) *
Buttons:
SEND RESET LINK
When the SEND RESET LINK button is clicked, the system must validate the email address. If the email is valid and exists in the system, a password reset link is sent to the user's email address, and a confirmation message is displayed: "A password reset link has been sent to your email address."
Business Rules:
BR01 - The Email field is mandatory.
Error message: "This field is required!"
BR02 - The user must enter a valid email address.
Error message: "Please enter a valid email address."
BR03 - The email address must be associated with an existing account.
Error message: "If this email address is in our database, we will send you a password reset link." (This is a common practice to avoid confirming which emails are registered).
